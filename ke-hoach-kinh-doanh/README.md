# Kế Hoạch Kinh Doanh

Thư mục này lưu trữ toàn bộ các bước triển khai của kế hoạch kinh doanh thực chiến, bao gồm phân tích chân dung khách hàng, giọng thương hiệu, cơ chế cốt lõi, lời chào hàng, kịch bản quảng cáo và các giao diện phễu tương tác.

Các file trong thư mục này sẽ được phân công cho các AI Agents chuyên biệt tạo lập nội dung chi tiết.
